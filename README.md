# BM Lab

Laboratório Estrutural da Lotofácil

## Objetivo

O BM Lab é uma plataforma destinada à automatização do Protocolo Banco Mestre v1.0.

Seu objetivo é registrar hipóteses, gerar jogos estruturais, auditar resultados e acumular evidências sobre o comportamento da Lotofácil de forma objetiva, reproduzível e auditável.
